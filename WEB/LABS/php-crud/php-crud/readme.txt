1. use sql\test-database.sql to create 'test' database and relevant tables. This example will only work if the database and tables are created.

2. use localhost\phpmyadmin to log into mysql databases and verify that test database has been created. 

3. place 'php-crud' folder in the web root folder (www folder)

4. Make sure to change database username and password in the config.php file, if you are using a different database credentials

5. use localhost\php-crud\index.php to view index page in the browser
